Sobre a ferramenta

O Validador de CEP em Massa é uma ferramenta gratuita desenvolvida para facilitar a consulta e validação de múltiplos CEPs de uma só vez.

Os dados de endereço são fornecidos pela API ViaCEP, um serviço público e gratuito. Inconsistências nos dados retornados devem ser reportadas diretamente ao ViaCEP.

Nenhum CEP consultado é armazenado em nossos servidores.

Tempo de resposta: respondemos e-mails em até 2 dias úteis. Para problemas com dados de endereço, verifique diretamente em viacep.com.br, pois a base de dados é mantida por eles.
